Folgen und Reihen

In der Mathematik behandelt man gelegentlich Konstrukte, die nach einer Regel strukturiert sind und in einer Reihenfolge aufgelistet, summiert, multipliziert oder anders verarbeitet werden. Wenn man die Elemente auflistet nennt man das eine Folge, wenn man die Elemente dabei noch addiert, nennt man das eine Reihe. Zum Beispiel:

Folge der geraden Zahlen: 2, 4, 6, 8, 10, ...
Folge der Primzahlen: 2, 3, 5, 7, 11, ...
eine Reihe: 1 + 2 + 3 + 4 + ...
noch eine Reihe: 2 + 3 + 5 + 7 + 11 + ...

Falls Sie schon programmieren können, würden Sie so etwas mit einer For-Schleife programmieren. Allerdings müssen Schleifen in der Programmierung immer nach endlich vielen Schritten aufhören, in der Mathematik können sie auch unendlich sein. Folgende Schreibweisen gibt es:

$\sum_{i=1}^{10} i^2$ bedeutet, dass von i=1 bis i=10 die Zahlen $i^2$ aufsummiert werden, also:

$\sum_{i=1}^{10} i^2 = 1 + 4 + 9 + 16 + 25 + 36 + 49 + 64 + 81 + 100$

Eine Gauss-Anekdote

Manchmal gibt es Formeln, mit denen man Reihen berechnen kann, ohne alle einzelnen Werte zu addieren. Der berühmte Braunschweiger Mathematiker Carl Friedrich Gauss hat diese Formel angeblich schon als Schulkind entdeckt:

$\sum_{i=1}^{n} i = 1 + 2 + 3 + ... + n = \frac{n(n + 1)}{2}$

Die Gauss-Anekdote besagt, dass der Lehrer seine Ruhe haben wollte und daher den Schulkindern die Aufgabe gab, die Zahlen von 1 bis 100 zu addieren. Der kleine Gauss war damit schon nach ein paar Minuten fertig, da er sich überlegte, dass man die Zahlen so anordnen kann, dass man die erste mit der letzten addiert, die zweite mit der vorletzten und so weiter:

$1 + 2 + 3 + \ldots + (n-2) + (n-1) + n = $
$ (1 + n) + (2 + (n-1)) + (3 + (n-2)) + \ldots = $
$ (1 + n) + (1 + n) + (1 + n) + \ldots = \frac{n}{2}(1 + n)$

Alternativ kann man das auch geometrisch beweisen:

Und hier ist noch eine weitere Reihe, für die es eine Formel und einen geometrischen Beweis gibt:

$\sum_{i=1}^{n}2i +1 = n^2$

Aufgaben

1. Schreiben Sie $\sum_{i=1}^{10} 2i+1$ als Summe aus, also ohne das Summenzeichen. (Das müsste Ihnen helfen, den geometrischen Beweis zu verstehen.)

2. Schreiben Sie $x_1 + x_2 + x_3 + x_4$ mit Summenzeichen.

Produkte

Statt addieren kann man auch wiederholt multiplizieren:

$\prod_{i=1}^{10} i^2 = 1 * 4 * 9 * 16 * 25 * 36 * 49 * 64 * 81 * 100$

Wenn es sich dabei um ein Produkt der ersten n natürlichen Zahlen handelt, so wird das auch als Fakultät bezeichnet und als n! geschrieben:

$n! = 1 * 2 * 3 * 4 * ... * n = \prod_{i=1}^{n} i$

Binomialkoeffizient

Die Fakultät wird auch zur Definition des Binomialkoeffizienten benötigt:

$\left(\begin{array}{c}n\\k\\\end{array}\right) = \frac{n!}{k!(n-k)!}$

Der Koeffizient wird in der Statistik verwendet und als "n über k" ausgesprochen. In Python kann man den Koeffizienten mit $comb(n,k)$ berechnen.

Aufgabe

Berechnen Sie $\left(\begin{array}{c}20\\18\\\end{array}\right)$, $\left(\begin{array}{c}20\\2\\\end{array}\right)$, $\left(\begin{array}{c}9\\3\\\end{array}\right)$, $\left(\begin{array}{c}9\\6\\\end{array}\right)$, $\left(\begin{array}{c}17\\16\\\end{array}\right)$, $\left(\begin{array}{c}17\\1\\\end{array}\right)$ und $\left(\begin{array}{c}17\\17\\\end{array}\right)$
