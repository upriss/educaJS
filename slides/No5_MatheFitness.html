Verstehen

Der Satz vom Nullprodukt besagt:

ab = 0 ⇔ a = 0 oder b = 0
Wenn x1 Lösung einer Gleichung x2 + px + q = 0 ist, folgt aus x = x1 dass: x - x1 = 0. Das Gleiche gilt für die andere Lösung x2. Wegen
x2 + px + q = 0 und (x - x1)(x - x2) = 0 gilt x2 + px + q = (x - x1)(x - x2).

Man kann weiter umformen: (x - x1)(x - x2) = x2 - x1x - x2x + x1x2. Also x2 + px + q = x2 - x1x - x2x + x1x2 und daher x1 + x2 = -p und x1x2 = q. Ganzzahlige Lösungen kann man also einfach bestimmen, indem man sich die Teiler von p anschaut.

Ist diese Argumentation richtig oder falsch? Geben Sie t für richtig (true) oder f für falsch ein.
