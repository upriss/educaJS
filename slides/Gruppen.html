Gruppeneigenschaften

Eine Menge mit einer binären Operation, die die folgenden Eigenschaften erfüllt

  • abgeschlossen
  • assoziativ
  • neutrales Element
  • inverse Elemente
ist eine Gruppe.

Wenn die Gruppe außerdem noch kommutativ ist, wird sie kommutative oder abelsche Gruppe genannt.

Abgeschlossen

Eine Menge G mit einer Operation ⋆ ist abgeschlossen, wenn
für alle Elemente a, b ∈ G gilt: a ⋆ b ∈ G.

Assoziativ

Eine Menge G mit einer Operation ⋆ ist assoziativ, wenn
für alle a, b, c ∈ G gilt: a⋆(b⋆c) = (a⋆b)⋆c.

 

Das kann man nicht an der Gruppentafel sehen. Man muss alle Tripel überprüfen. Allerdings folgt aus der Assoziativität die Sudoku-Eigenschaft (jedes Element steht genau einmal in jeder Spalte/Zeile).

Neutrales Element

Eine Menge G mit einer Operation ⋆ hat ein neutrales Element n, wenn für alle a ∈ G gilt: a⋆n = n⋆a = a.

Inverse Elemente

Eine Menge G mit einer Operation ⋆ hat inverse Elemente , wenn es für alle a ∈ G ein a-1 gibt mit a⋆a-1 = a-1⋆a = n
(wobei n das neutrale Element ist).

Kommutativ

Eine Gruppe G mit einer Operation ⋆ heißt kommutativ (oder abelsch), wenn für alle a,b ∈ G gilt: a⋆b = b⋆a.

Wozu braucht man Gruppen?

Wenn man weiß, dass eine Menge mit einer Operation eine Gruppe bildet, dann weiß man, welche Rechenregeln gelten.

"Gruppe" ist eine algebraische Struktur.

In der Mathematik reicht es oft zu wissen, mit welcher Struktur man es zu tun hat, damit man weiß, wie man damit rechnen kann.
