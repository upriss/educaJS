Examples of using the educaJS software for displaying concept lattices. The interactive page can be linked to or embedded in an iframe (as below). A link to the current state of a page is generated and displayed in the web developer console from where it can be copied. The software is in an alpha state with some bugs.

Examples: Live in Water lattice  Person/adult/female/male lattice  Nominal scale  Ordinal scale  Interordinal scale
